# 周期建议 · 寒武纪方法论 AI 选股系统

> **A股资源类国企自动筛选器** — 基于寒武纪方法论的五级漏斗投资体系

## 📋 项目概述

将寒武纪鳄鱼的价值投资方法论转化为可执行的 Web 应用，涵盖：

- **五级漏斗筛选**：企业性质 → 行业筛选 → 基本面量化 → 周期判断 → 估值确认
- **自动选股引擎**：对 A 股商品/资源类国企进行自动筛选评分
- **AI 提示词系统**：9 组可直接在 ChatGPT/Claude 等 AI 中使用的投资分析指令
- **投资建议**：基于评分自动生成买入/关注/排除建议

## 🎯 核心策略

| 维度 | 标准 |
|------|------|
| 投资标的 | 仅央企/省国资委控股的资源类商品企业 |
| 买入时机 | 全产业链亏损 + PE≤15× + 拐点出现 |
| 业绩爆发 | 3年净利润弹性≥5倍 |
| 安全边际 | 国企在行业低谷享有国家保护 |
| 卖出信号 | 行业狂热 + PE>60× + 产业资本撤退 |

## 🔍 五级漏斗筛选引擎

### 关1 · 企业性质过滤（硬性条件）
- ✓ 央企（国务院国资委）
- ✓ 省级国资委控股
- ✓ 地市级国资委控股
- ✗ 民营/外资/管理层控股 → 直接排除

### 关2 · 行业筛选（硬性条件）
- ✓ 战略矿产（稀土、钨、锑、钼、铀、钴、锂等）
- ✓ 基本金属（铜、铝、铅、锌、锡等）
- ✓ 能源（煤炭、石油、天然气、铀）
- ✓ 化工（磷化工、钾肥等）
- ✗ AI/芯片/券商/消费品 → 直接排除

### 关3 · 基本面量化
- 3年净利润弹性 ≥ 5×
- 毛利率 ≥ 20%
- 经营现金流/净利润 > 0.8
- 资产负债率 < 65%

### 关4 · 周期位置
- 产品价格处于近5年低位 (< 40% 分位) 最佳
- 全产业链亏损信号

### 关5 · 估值确认
- PE ≤ 15×（买入区间）
- 正常化利润 PE < 10×（周期底部用）

## 📦 数据说明

1. 通过 EdgeOne Functions 代理实时行情 API
2. 连接 Wind/东方财富等数据源
3. 使用 AI 搜索（Perplexity/Claude）获取基本面数据
